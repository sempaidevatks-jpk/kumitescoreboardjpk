Table Official Quick Guide

Main Controls
  • Click main timer – Edit remaining seconds.
  • Start / Pause button or Space – Start / stop timer.
  • Reset – Reset timer to selected duration.
  • Reset Match – Clears scores, penalties, senshu and timer.
Keyboard – Scores
  • A + 1 / 2 / 3 – AKA +1 / +2 / +3.
  • B + 1 / 2 / 3 – AO +1 / +2 / +3.
  • A + S – Toggle senshu for AKA.
  • B + S – Toggle senshu for AO.
Keyboard – Penalties
  • A + C – AKA C1 → C2 → C3 → clear.
  • B + C – AO C1 → C2 → C3 → clear.
  • A + V – AKA HC on/off.
  • B + V – AO HC on/off.
  • A + H – AKA Hansoku.
  • B + H – AO Hansoku.
Other Shortcuts
  • R – Reset main timer (when paused).
  • Esc – Close this help window.
All key actions are logged in the Event Log on the right, so you can review every change if there is a protest or query.[web:24]
